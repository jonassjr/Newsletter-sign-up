# Newsletter sign up

Página web para assinatura de newsletter

## Tecnologias Utilizadas
- React
- JavaScript
- HTML
- Tailwind CSS
- Git

## Funcionalidades
- Cadastro de usuários: permite que os usuários se cadastrem para receber a newsletter, inserindo seu endereço de e-mail.
